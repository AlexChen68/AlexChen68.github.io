## Spring 的表达式语言 (SpEL)

Spring Expression Language (简称 "SpEL") 是一种强大的表达式语言，支持在运行时查询和操作对象。语言语法类似于 Unified EL，但提供了其他功能，最有名的是方法调用和基本字符串模板功能。

表达式语言支持以下功能:

- 文字表达
- B 布尔和关系运算符
- 正则表达式
- 类表达式
- 访问属性，数组，list 和 maps
- 方法调用
- 关系运算符
- Assignment
- 调用构造器
- bean 的引用
- 数组的构造
- 内嵌的 list
- 内嵌的 map
- 三元表达式
- 变量
- 用户自定义函数
- 集合映射
- 集合选择
- 模板表达式
