# InfluxDB 介绍

## InfluxDB 是什么？

InfluxDB 是一个由 InfluxData 开发的开源**时序型数据库**。它由 Go 语言开发，着力于高性能地查询与存储时序型数据。

InfluxDB 被广泛应用于**存储系统的监控数据**，**IoT 行业的实时数据**等场景。
