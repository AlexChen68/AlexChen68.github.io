## 预期

- 使用 Zookeeper 实现分布式
- 使用 Hadoop HDFS 实现分布式存储
- 使用 Flink 实现计算引擎

## 准备工作

- 需要准备最少三台机器，来搭建一个最少节点的集群；三台机器打通网络，互相配置 ssh 免密登录。
- 准备需要部署的软件及依赖环境的安装包：
  - JDK8 安装包 jdk-8u381-linux-x64.rpm
  - Zookeeper 安装包 zookeeper-3.3.2.tar.gz
  - Hadoop 安装包 hadoop-3.3.6.tar.gz
